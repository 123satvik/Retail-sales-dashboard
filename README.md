# Retail Sales Analytics Dashboard

## Overview
This project analyzes **30,000+ rows of retail sales data** to identify sales trends, product performance, and supplier contributions. The goal is to transform raw retail data into meaningful business insights.

## Tools Used
- Excel – Data cleaning and preprocessing  
- SQL – Data querying and analysis  
- Python (Jupyter Notebook) – Exploratory Data Analysis  
- Pandas, Matplotlib, Seaborn – Data manipulation and visualization  
- Power BI – Interactive dashboard creation  

## Key Features
- Data cleaning and preprocessing  
- Sales trend analysis  
- Product category performance analysis  
- Supplier performance insights  
- Interactive Power BI dashboard  

## Dataset
The dataset contains **30,000+ retail transaction records** including product details, suppliers, and sales information.

## Outcome
Developed an interactive dashboard and performed analysis to understand **sales performance, top products, and supplier contributions**.
